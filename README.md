# Array Challenge

## Challenge 1

**Instructions:**

> Use some of the array methods that we've looked at to mutate the following array to the expected results below.
> Reverse the order of the array.

```js
const arr = [1, 2, 3, 4, 5];
```

**Expected results:**

```js
console.log(arr); // [6, 5, 4, 3, 2, 1, 0];
```
